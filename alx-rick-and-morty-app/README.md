
# alx-rick-and-morty-app

A Next.js application using TypeScript, Tailwind CSS, Apollo Client, and GraphQL to interact with the Rick and Morty API.


## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.

## Project Structure
- `graphql/` — Apollo Client setup and GraphQL queries
- `pages/_app.tsx` — App entry with ApolloProvider

## Features
- Next.js with TypeScript
- Tailwind CSS for styling
- Apollo Client for GraphQL
- Rick and Morty API integration
